# Legend-of-Devslopes
A 3d fantasy melee game made from following Devslopes course on Unity Game Development

Features include:
- Navmesh agents
- Raycasts
- Enemy pathfinding
- Enemy spawning 
- Minimap
- Pause menu
- Start menu
- Health powerups

Bugs: 
- Tanker hurt animations not playing
- Tanker attacks don't register
